# UX/UI 文档版本记录

- v0.1 初始化模块划分与目录映射
- v0.2 (2026-01-13) 布局与交互优化：
  - **Sash 交互修复**：修复分割线拖拽时的坐标跳变问题，采用逻辑中心 (`props.left`) 计算偏移量，确保零跳变。
  - **布局约束放宽**：编辑器最小宽度下调至 200px，移除最大宽度限制；预览区最大宽度提升至 1500px，提升多列布局自由度。
  - **默认沉浸式体验**：Markmap 预览面板与悬浮编辑面板默认隐藏，应用启动默认进入单栏编辑模式。
  - **视觉洁癖**：悬浮面板容器在关闭状态下不再渲染空 DOM，消除“幽灵白色背景”。
- v0.3 (2026-01-16) 侧边栏架构重构与交互升级：
  - **统一侧边栏体验**：重构为 `UnifiedSidebarShell`，实现 Fixed 和 Overlay 模式的单实例无缝切换。
  - **智能状态同步**：无论以推挤（Fixed）还是悬浮（Overlay）模式呼出侧边栏，其宽度始终保持一致，遵循用户最后一次调整的值。
  - **动态图标反馈**：侧边栏切换按钮现在会根据状态显示 `PanelLeftOpen`（展开提示）或 `PanelLeftClose`（收起提示），提供明确的视觉引导。
